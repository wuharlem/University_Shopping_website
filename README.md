# University_Shopping_website NTU SHOPPING 出清台大

此專案為**台科大網頁製作**課堂作業。出發點是因為[出清台大](https://www.facebook.com/groups/closeout/)是以台大為中心，附近學生會加入的拍賣臉書社團，因為使用者眾多，臉書使用交易不便，因此設計出清台大網站版本，登入介面以台大、師大、台科大，出清台大主體學校為背景。
